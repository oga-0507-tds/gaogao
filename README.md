# Gaogao Game 🐕💢

眠っている番犬を起こさないように、そっと骨を取っていくスリル満点のパーティーゲームです！
スマホやPCのブラウザで手軽に遊べます。

## 🎮 遊び方

1.  **プレイヤー登録**: 参加する人数と名前を設定します。
2.  **骨を選ぶ**: 自分のターンになったら、好きな色の骨を選んでタップ（クリック）します。
    *   ⚪ **White**: 1ポイント（安全め）
    *   🔵 **Blue**: 3ポイント（少し回数が必要）
    *   🔴 **Red**: 5ポイント（回数が多く高リスク！）
3.  **ゲームオーバー条件**:
    *   **ハズレの骨**を引いてしまう（確率で起きます）。
    *   **連打**しすぎて犬を怒らせてしまう（怒りゲージMAX）。
    *   **制限時間（10秒）**が過ぎてしまう。
4.  **勝敗**: 犬を起こしてしまった人が負け！それまでに獲得したスコアでランキングが決まります。

## 🚀 起動方法

Node.jsがインストールされている環境であれば、プロジェクトのルートディレクトリで以下のコマンドを実行してください。

```bash
npx serve .
```

コマンド実行後、ターミナルに表示されるURL（例: `http://localhost:3000`）にブラウザでアクセスするとゲームが始まります。

## 📂 ファイル構成

*   `index.html`: ゲームのエントリポイント
*   `js/`: スクリプトファイル
    *   `game.js`: ゲームの進行管理（Controller）
    *   `gameService.js`: ゲームのロジック・判定（Model）
    *   `components/`: 各画面のUIパーツ（View）
        *   `init.js`: タイトル画面
        *   `name-input.js`: 名前入力画面
        *   `playing.js`: プレイ画面（メイン）
        *   `game-over.js`: 結果発表画面
*   `src/`: 静的リソース
    *   `style.css`: スタイルシート
    *   `img/`: 画像リソース
